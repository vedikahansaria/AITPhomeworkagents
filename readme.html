Customer Churn Prediction Model

This project implements a Machine Learning pipeline to predict customer churn. It analyzes historical customer data to identify patterns indicating which customers are likely to stop using the service.

Goal: To provide actionable insights for retention strategies by classifying customers into 'Churn' vs. 'No Churn'.

Prerequisites

  • Python 3.8 or higher
  • pip (Python package installer)

Installation

1. Clone or download this repository.

2. Install the required dependencies using the command below:

pip install -r requirements.txt

How to Run

Ensure your dataset (e.g., customer_data.csv) is in the root directory. Run the main script:

python main.py

Key Features

  • Data Preprocessing: Handling missing values and encoding categorical variables.
  • EDA: Visualization of churn distribution and correlation matrices.
  • Modeling: Implementation of Logistic Regression (or Random Forest).
  • Evaluation: Output of Accuracy, Precision, Recall, and Confusion Matrix.
